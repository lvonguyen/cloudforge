# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via email:

**Email:** liem@vonguyen.io

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

---

## [!] Important

These are portfolio/demonstration projects. They are **not intended for production use** without additional security hardening.

Do not use these projects to process real sensitive data without proper review and customization.
